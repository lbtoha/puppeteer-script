# How to run this script

## Clone this Repository

```
https://github.com/lbtoha/puppeteer-script.git
```
## Go to 
```
cd puppeteer-script
```

## Now install node_modules by running this command
 ```
 npm install
 ```

 #Now Run this command
 ```
 npm start
 ```
